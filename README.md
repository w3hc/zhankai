# Zhankai

Writes the content of all files of a given repo. 

## Use

### Install

```bash
npm install -g zhankai
```

### Verify

```bash
zhankai --version
```

### Run

```bash
zhankai
```

A `zhankai_output.md` is created at the root of your repository.

## Contrib

### Install

```bash
pnpm i
```

### Run

```bash
pnpm build
npm run install -g .
zhankai
```

## Versions

- pnpm v8.7.5
- node v20.9.0
